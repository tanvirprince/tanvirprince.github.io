# Tanvir Rahman Prince — Portfolio Website

A responsive, animated portfolio built with HTML5, Bootstrap 5, custom CSS and vanilla JavaScript.

## Run locally

Open `index.html` directly, or use a local server:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Customize

- Main content: `index.html`
- Design and responsive styles: `assets/css/style.css`
- Theme, typewriter and scroll animations: `assets/js/main.js`
- Profile image: `assets/images/tanvir-prince.webp`
- Downloadable resume: `assets/files/Tanvir_Rahman_Prince_CV.pdf`

## Deployment

This is a static site and can be deployed to GitHub Pages, Netlify, Vercel, Cloudflare Pages or any standard web host.
